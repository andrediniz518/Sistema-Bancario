## Sistema Bancario


Sistema desenvolvido com o intuito de praticar lógica de programação,
modulizarização de projeto, funções parametros e retorno de valor,
manipulação de lista e tomadas de decisões.

Este Projeto simula operações básicas de uma conta bancaria, permitindo realizar
depositos, saques e consulta de extrato.


funcionalidades:

- menu interativo com usuario
- depositar
- sacar
- mostrar extrato
- mostrar saldo


estrutura do projeto:

- main.py - controla o fluxo principal do programa
- lib\banco.py - onde está regras de negócios, sacar, depositar...
- lib\menu.py - interface amigavel para o usuario acessar o banco.


tecnologia utilizadas

- Python 3


como executar

1. abaixe os arquivo do projeto
2. execute o arquivo main.py
